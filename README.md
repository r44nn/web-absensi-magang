📌 Deskripsi Singkat

Aplikasi Web Absensi membantu perusahaan mencatat dan mengelola data kehadiran karyawan secara digital, efisien, dan terintegrasi.

Frontend: React.js + Tailwind CSS

Backend: Node.js + Express.js + MongoDB
